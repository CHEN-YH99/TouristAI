# 🎉 欢迎使用 TouristAI！

## 你好！👋

欢迎来到TouristAI - AI智能旅游攻略生成平台！

这是一个基于AI大语言模型的智能旅游规划助手，可以帮你快速生成个性化的旅游攻略。

## ✨ 你可以做什么？

### 💬 与AI对话
只需告诉AI你的旅行计划，它就能为你生成详细的攻略：
- "我想去日本玩5天，预算1万元"
- "推荐一些适合亲子游的地方"
- "成都有什么必吃的美食？"

### 📝 保存攻略
生成的攻略可以一键保存，随时查看：
- 完整的行程规划
- 景点推荐
- 美食建议
- 预算分配

### 🎯 个性化定制
根据你的偏好定制专属攻略：
- 旅行天数
- 预算范围
- 兴趣爱好
- 出行人数

## 🚀 快速开始

### 第一步：启动项目

#### Windows用户
```bash
start.bat
```

#### Mac/Linux用户
```bash
./start.sh
```

### 第二步：访问应用
打开浏览器访问：http://localhost:3000

### 第三步：注册账号
1. 点击"注册"按钮
2. 填写邮箱、用户名和密码
3. 完成注册

### 第四步：开始对话
1. 登录后进入对话页面
2. 输入你的旅行需求
3. 等待AI生成攻略
4. 保存你喜欢的攻略

## 🎓 使用技巧

### 1. 提供详细信息
告诉AI更多细节，获得更精准的建议：
- 旅行天数
- 预算范围
- 兴趣偏好
- 特殊需求

### 2. 多轮对话
可以继续追问，细化你的需求：
- "能详细说说第三天的安排吗？"
- "有没有更便宜的住宿选择？"
- "推荐一些当地特色美食"

### 3. 保存攻略
生成满意的攻略后记得保存：
- 点击"保存攻略"按钮
- 填写标题和标签
- 随时在"我的攻略"中查看
